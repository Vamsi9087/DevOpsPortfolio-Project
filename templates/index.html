About This Project

This project is a complete DevOps portfolio application that demonstrates end-to-end development and deployment of a Flask-based web application. It includes a dynamic frontend with custom CSS animations, a Python Flask backend for handling user comments, persistent data storage using SQLite, and full containerization using Docker.

The application is deployed using Docker Compose and automated through a Jenkins CI/CD pipeline that builds, tests, and deploys the application. This project reflects real-world DevOps practices such as containerized deployments, automation, and production-ready application design.

Tools Used

  • Python & Flask – Backend development
  • Docker & Docker Compose – Containerization
  • Jenkins – CI/CD automation
  • HTML, CSS, JavaScript – Frontend & animations
  • Git & GitHub – Version control
